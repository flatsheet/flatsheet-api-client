Flatsheet(options, options.host, options.username, options.password)

Flatsheet API Client

parameter type description
options Object

options for the api client
options.host String

flatsheet server host
options.username String

username, only needed if making POST, PUT, DELETE requests
options.password String

password, only needed if making POST, PUT, DELETE requests

Examples

var flatsheet = require('flatsheet-api-client');
var client = flatsheet();
